ВОВЕД ВО UX UI ДИЗАЈН
Овој модул претставува целосен вовед во светот на дигиталниот дизајн. Ќе ги научиш разликите меѓу UX и UI, процесот на креирање кориснички пат, wireframes, прототипи и како дизајнот влијае на бизнис резултатите.
HTML
Практичен модул за разбирање како функционираат веб-страниците. Ќе научиш основи на HTML за да разбереш структура, елементи, layout-логика и како твоите UI дизајни се претвораат во реални веб-страници
CSS
Во овој модул ќе ја совладаш визуелната страна на front-end основите. Ќе научиш стилови, типографија, боја, spacing, responsiveness и како CSS се користи за да се оживее UI дизајнот што го креираш.
ADOBE PHOTOSHOP
Ќе научиш обработка на фотографии, ретуширање, креирање графички елементи, mockups и подготовка на материјали. Photoshop останува клучна алатка за визуелна продукција во UI дизајн.
ADOBE ILLUSTRATOR
Модул фокусиран на креирање скалабилна графика, icons, логоа, векторски елементи и илустрации. Illustrator ти овозможува да креираш чисти, професионални UI assets за мобилни и веб апликации.
FIGMA
Најкористената алатка за UX/UI. Ќе научиш како да креираш wireframes, интерфејси, компоненти, auto-layout, responsive screens, интерактивни прототипи и колаборација со девелопери. Практични задачи и реални сценарија се дел од секој час.
USER EXPERIENCE (UX)
Длабински модул каде ќе ги совладаш истражувачките и стратегиските процеси: user research, personas, user flows, информациска архитектура, usability testing и UX writing. Целта е да креираш производи што решаваат реални проблеми.
USER INTERFACE (UI)
Фокус на визуелниот дел од дизајнот: боја, типографија, композиција, визуелна хиерархија, дизайн системи, компоненти и UI patterns. Ќе научиш да градеш професионални, современи и функционални интерфејси за веб и мобилни апликации.
